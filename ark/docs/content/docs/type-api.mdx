### toJsonSchema

Each `Type` instance exposes a `toJsonSchema()` method that can be used to generate a corresponding [JSON Schema](https://json-schema.org/).

```ts
const User = type({
	name: "string",
	email: "string.email",
	age: "number >= 18"
})

const schema = User.toJsonSchema()
// {
//   type: "object",
//   properties: {
//     name: { type: "string" },
//     email: { type: "string", format: "email" },
//     age: { type: "number", minimum: 18 }
//   },
//   required: ["name", "email", "age"],
//   $schema: "https://json-schema.org/draft/2020-12/schema"
// }
```

Options can be passed to change the behavior including how incompatibilities are handled. See [the associated config docs](/docs/configuration#toJsonSchema) for more details.
