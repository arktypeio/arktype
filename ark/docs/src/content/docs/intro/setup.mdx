## Install

<Tabs>
	<TabItem label="pnpm" icon="pnpm">
		<Code code="pnpm install arktype" lang="bash" />
	</TabItem>
	<TabItem label="npm" icon="seti:npm">
		<Code code="npm install arktype" lang="bash" />
	</TabItem>
	<TabItem label="yarn" icon="seti:yarn">
		<Code code="yarn install arktype" lang="bash" />
	</TabItem>
	<TabItem label="bun" icon="bun">
		<Code code="bun install arktype" lang="bash" />
	</TabItem>
</Tabs>

You'll also need...

- TypeScript version `>=5.1`
- A `tsconfig.json` with [`strict`](https://www.typescriptlang.org/tsconfig#strict) or [`strictNullChecks`](https://www.typescriptlang.org/tsconfig/#strictNullChecks) enabled
