# intersection

## operator

-   [&](./intersection.md)

## string

-   "L&R"

## tuple

-   [L, "&", R]

## helper

-   intersection(L,R)

## example

-   const intersection = type("/@arktype\.io$/ & email") <br/>
-   const tupleIntersection = type(["string", "&", "uppercase"]) <br/>
-   const helperIntersection = intersection("string","uppercase") <br/>
